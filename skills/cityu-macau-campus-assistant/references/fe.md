## 1. 学院概况与课程体系

### 1.1 学院定位

澳门城市大学教育学院（School of Education，简称 SOE）成立于 **2014年**，是澳门城市大学专注于研究生教育的学院。

**核心特点**：
- **专注研究生教育**——学院自成立之日起不设本科课程，仅开设硕士和博士研究生课程；
- 每年经过严格挑选与审核，招收约 **150名硕士** 及 **60名博士** 研究生；
- 教学与学术研究并重发展，致力于为澳门及周边地区培养高层次教学、科研和教育管理人才；
- 以研究作指导，以科研促教学，教书育人重在育人。

### 1.2 学院面向学生的核心卖点

1. **专注研究生教育**：不设本科，所有资源集中于硕士和博士培养；
2. **多元化师资**：以专兼结合方式邀请国内外知名学者担任特聘教授；
3. **科研平台完善**：设有澳门全球教育研究与创新研究所（IMED）、人工智能教育研究创新实验室等；
4. **国际化视野**：课程强调国际视野和跨文化交流能力培养；
5. **学科排名提升**：教育学院科研教学双丰收，学科排名首登 THE 全球榜单。

### 1.3 课程体系一览

| 层级 | 课程名称 | 简称 | 学制 | 授课语言 |
|---|---|---|---|---|
| 硕士 | 教育学硕士学位课程 | Med | 2 年（全日制） | 中文辅以英语 |
| 硕士 | 教学研究硕士学位课程 | MTLR | 2 年（全日制） | 中文辅以英语 |
| 博士 | 教育学博士学位课程 | PhD in Education | 3 年（全日制） | 中文辅以英语 |
| 博士 | 教育博士学位课程 | EdD | 待定 | 待定（待政府刊宪后始行招生） |

> ⚠️ **特别提示**：教育学院所有课程均**未包含澳门中小幼教师专业资格培训内容**。

## 2. 学生最关心问题总览：按层级速查

### 2.1 硕士、博士关键差异

| 项目 | 教育学硕士 Med | 教学研究硕士 MTLR | 教育学博士 PhD in Ed |
|---|---|---|---|
| 学制 | 2 年 | 2 年 | 3 年 |
| 总学分 | 39 学分 | 33 学分 | 50 学分 |
| 专业方向 | 3 个：教育行政管理、终身学习、教育资讯技术 | 2 个：课程与教学、资讯技术与教学 | 不分方向，但选修课3选1 |
| 毕业论文/项目 | 学位论文（12学分）+ 答辩 | 项目报告（9学分，行动研究/教学实务型） | 学位论文（40学分，不少于5万字）+ 答辩 |
| 入学研究计划要求 | 无明确要求 | 无明确要求 | 报名时须提交不少于3,000字研究计划 |
| 授课语言 | 中文辅以英语 | 中文辅以英语 | 中文辅以英语 |
| 授课形式 | 面授 | 面授 | 面授 |

### 2.2 对学生利益影响最大的规则

1. **入学年份规则优先**：课程结构、毕业要求和学分要求应以学生本人入学年度的课程计划/学生手册为准。
2. **论文 vs 项目报告**：Med 要求学位论文+答辩；MTLR 用项目报告替代原创论文（行动研究或教学实务型），这是澳门高校中独一无二的特色。
3. **博士申请需研究计划**：PhD in Education 报名时须提交不少于3,000字研究计划。
4. **所有课程大纲以政府宪报为准**：大学保留权利取消开办报读人数不足的科目。
5. **不含教师专业资格培训**：教育学院所有课程均未包含澳门中小幼教师专业资格培训内容。
6. **往届内地生需考研成绩**：往届本科毕业生报考硕士需参加全国硕士研究生招生考试且初试总成绩过国家A类线。

## 3. 入学与招生：学生应关心什么

### 3.1 硕士入学要求

#### 共同学历要求

| 学生类别 | 学历要求 |
|---|---|
| 本地生 | 学士学位/应届学士毕业或相等程度 |
| 内地生 | 教育部认证的本科学历（学信网可查询）；国(境)外毕业须教育部留学服务中心网可查 |
| 香港、台湾地区及国际学生 | 学士学位或同等学历 |

#### 往届生附加要求

| 学生类别 | 附加要求 |
|---|---|
| 应届本科毕业生 | 报考所有硕士课程均不受考研限制 |
| 往届本科毕业生（内地生） | 需参加2026年全国硕士研究生招生考试且初试总成绩过国家A类线 |

#### 专业背景偏好

- 专业不限，具**教育相关专业背景优先**。

#### 录取原则

大学根据**面试成绩**，综合以下因素择优录取：
- 专业背景
- 工作经历
- 学术成果
- 获奖经历
- 前置学历成绩
- 前置学历院校层次
- 英文能力

### 3.2 博士入学要求

#### 教育学博士（PhD in Education）

## 5. 教育学硕士学位课程（Med）

### 5.1 课程总览

| 项目 | 详情 |
|---|---|
| 课程名称 | 教育学硕士学位课程 |
| 学术范畴 | 教育学 |
| 修读模式 | 全日制 |
| 正常修业期限 | 两年 |
| 授课形式 | 面授 |
| 授课语言 | 中文辅以英语 |
| 授课时数 | 每学分授课15小时 |

### 5.2 课程特色

- 专注于提升学生在高等和基础教育领域所需的知识、技能及专业态度；
- 弥补澳门及周边地区高层次教育人才的不足；
- 全面学习教育学基础理论知识与技能，向多个细分专业方向拓展；
- 毕业生可增强职业竞争力，成为与社会发展相适应的教育研究中坚人才。

### 5.3 课程目标

1. 培育课程与教学研究的优秀人才；
2. 培育资讯科技研究的优秀人才；
3. 培育成人教育/终身学习领域的教学与管理人才。

### 5.7 毕业要求

1. 完成9门科目并取得 **27学分**；
2. 完成毕业论文，并通过硕士学位论文的答辩，取得 **12学分**；
3. 没有任何欠费；
4. 遵守本大学的规则。

### 5.8 预期学习成果

1. 应用课程与教学领域之专业技能，进行文献与实务的省思与批判；
2. 统整课程与教学理论与实务技能，进行研究、思考与批判；
3. 探究课程与教学议题并进行论文发表；
4. 对课程与教学的研究发现能进行创新与实践。

### 5.9 毕业生出路

1. 各大专院校教职，从事学术研究及教学工作；
2. 各级教育研究机关之研究人员或行政人员；
3. 学校行政管理工作；
4. 各机关教育培训专业人员。

### 6.2 课程特色

1. 培养具有较强의科研、教学和为课程与教学实践服务的人才；
2. 培养具有扎实的专业理论基础、较强的现代教育研究方法知识和能力的人才；
3. 以"教与学"作为主要研习对象，培养具有较强教学实践能力的人才。

# 第二部分：博士课程详解

## 8. 教育学博士学位课程（PhD in Education）

### 8.1 课程总览

| 项目 | 详情 |
|---|---|
| 课程名称 | 教育学博士学位课程 |
| 学术范畴 | 教育学 |
| 修读模式 | 全日制 |
| 正常修业期限 | 三年 |
| 授课形式 | 面授 |
| 授课语言 | 中文辅以英语 |
| 总学分 | 50学分 |
| 论文字数 | 不少于**五万字** |
| 入学研究计划 | 报名时须提交不少于**3,000字**研究计划 |

### 8.2 课程特色

- 旨在为澳门及周边地区培养具备最高学术研究能力的专业及领导人才；
- 通过实践多样的研究方法，学员可以沉浸于最新的科研项目、钻研最前沿的教育研究问题；
- 毕业生通常会进入高等教育机构、研究机构或政府部门，从事高层次教学、研究及管理工作。

### 8.3 课程目标

1. 培养学校营运之**研究与领导人才**；
2. 培养教育行政机关之**研究与行政人才**；
3. 培养教育事业之**研究与经营人才**。

### 8.4 课程发展方向与重点

1. **强大的师资队伍**：拥有一批在教育领域具有深厚学术造诣和丰富教学经验的教授和学者；
2. **前沿的研究方向**：涵盖教育技术、教育评估与测量、教育心理学等前沿研究方向；
3. **跨学科研究**：鼓励学生结合心理学、社会学、大数据科学等多学科知识；
4. **国际化的教育视野**：与国际知名高校和研究机构合作，提供国际化教育环境和研究机会。

### 8.5 学分结构

| 组成部分 | 学分 |
|---|---|
| 核心课程（必修） | 8 |
| 专业方向选修课（3选1） | 2 |
| 毕业论文 | 40 |
| **总计** | **50** |

### 8.6 核心课程（必修，8学分）

| 科目 | 学分 |
|---|---|
| 教育机构变革 | 2 |
| 进阶教育研究方法 | 2 |
| 全球化与教育的比较研究 | 2 |
| 博士论文准备 | 2 |

### 8.7 专业方向选修课（3选1，2学分）

| 科目 | 学分 |
|---|---|

| 项目 | 详情 |
|---|---|
| 学分 | 40 |
| 字数要求 | 不少于**五万字** |
| 答辩要求 | 通过原创论文的答辩 |

### 8.9 毕业要求

1. 完成5门科目并取得 **10学分**；
2. 通过原创论文的答辩并取得 **40学分**；
3. 没有任何欠费；
4. 遵守本大学的规则。

### 8.10 预期学习成果

1. 诊断及有效处理组织问题；
2. 运用网络资源进行学习与管理；
3. 进行独立研究与推动学术发展；
4. 拟定与推动组织经营计划；
5. 拟定与推动组织变革与创新方案；
6. 有效执行组织的沟通协调与行销。

### 8.11 毕业生出路

1. 各大专院校教职，专注于学术研究及教学工作；
2. 公务人员任用考试，成为各级教育行政机关的研究或行政人员；
3. 学校行政管理或教学工作；
4. 公私立教育研究机构教育企划及研究工作；
5. 各类教育事业的经营与管理领导人；
6. 企业人力资源管理及教育训练的专业人员。
