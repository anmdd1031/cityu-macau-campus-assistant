## 1. 荣誉班概况与培养体系

### 1.1 荣誉班定位

澳门城市大学**荣誉班（Honours Class）**于 **2024/2025学年**隆重推出，是澳门城市大学打造的**创新人才培养模式**的全新探索。

**核心特点**：
- **试点启动**：第一届以**数据科学学院**为试点单位；
- **招生范围**：在**计算机科学学士学位课程**和**智能科技与服务学士学位课程**的大一至大三学生中进行成员选拔；
- **第二阶段**：将拓展至**全校招生**（第二届荣誉班将面向全校学士学位课程招生，在大一至大四的本科生中进行成员选拔）；
- **合作单位**：澳门城市大学联合**深圳零一学院**共同打造；
- **培养模式参考**：基于**清华大学钱学森班15年培养经验**。

### 1.2 荣誉班面向学生的核心卖点

1. **顶尖师资团队**：由郑纬民院士、郑泉水院士、方滨兴院士、周万雷教授等顶尖学者授课；
2. **一对一导师指导**：每位学生都将获得城大或深圳零一学院优秀导师的一对一长期指导；
3. **进阶式科研训练**：全新设计的进阶式精深学习体系，分为初级科研训练、高阶研究训练、自主科研训练三个阶段；
4. **知名企业实习**：与腾讯、百度、阿里、华为等知名科技企业和学术单位建立深厚合作关系，为学生提供丰富实习岗位；
5. **前沿挑战性问题引领**：顶尖师资团队开设 **X-Challenge 课程**，与学生探讨前沿挑战性问题；
6. **探索交叉融合领域**：鼓励学生在人工智能、数据安全、类脑计算、金融科技、智慧城市、生物信息、超算等前沿领域自由探索；
7. **研究奖励资助体系**：经过选拔入选的学生将获得**长期的研究资助**；
8. **多元文化成长环境**：澳门作为东西方文化交汇点，融合中华文化与西方文化，经常举办各类国际性会议、论坛、展览。

### 1.3 培养目标

> 在国际化的学习环境中掌握前沿学科知识，成长为具有**创新思维**的**实用型高端人才**。

具体培养目标：
1. 培养创新思维和解决实际问题的能力；
2. 激发学生内生动力与热情；
3. 培养综合创新能力和实践能力；
4. 引导学生成为**新领域的先锋**。

## 6. 荣誉进阶研究课程体系

### 6.1 课程总览

荣誉进阶研究课程体系是荣誉班的**第二阶段**课程，旨在进行进阶式精深学习。

**课程模块**：分为三个阶段——**初级科研训练**、**高阶研究训练**、**自主科研训练**。

### 6.2 课程列表

| 编号 | 课程名称（中文） | 课程名称（英文） | 阶段 |
|---|---|---|---|
| 201 | 荣誉班--挑战性问题 | Honor Class X-Idea | 初级科研训练 |
| 202 | 荣誉班--大学生研究训练 | Honor Class SRT | 高阶研究训练 |
| 203 | 荣誉班--自主/海外研究训练 | Honor Class ORIC/SURF | 自主科研训练 |

### 6.3 课程详解

#### 201 挑战性问题（X-Idea）

- **课程目标**：由顶尖师资团队开设，与学生探讨前沿挑战性问题；
- **培养能力**：提出创新性研究问题、解决复杂挑战性问题的能力；
- **授课教师**：郑纬民院士、郑泉水院士、方滨兴院士、周万雷教授等。

#### 202 大学生研究训练（SRT）

- **课程目标**：在导师指导下进行系统性的科研训练；
- **培养能力**：科研方法、实验设计、数据分析、论文写作；
- **授课教师**：城大及深圳零一学院的菁锐导师。

#### 203 自主/海外研究训练（ORIC/SURF）

- **ORIC**：校内自主研究机会；
- **SURF**：通常指"暑期本科生研究奖学金"项目，涉及海外研究交流；
- **培养能力**：独立科研能力、国际视野、跨文化交流能力。
