## 1. 学院概况与课程体系

### 1.1 学院定位

澳门城市大学大健康学院（Faculty of Health，简称 FH）成立于 **2024年3月28日**，是**全国乃至全球首家以大健康命名的专业教育和学术机构**。学院以培养大健康产业所需的专业人才为目标，秉持"服务澳门、融入湾区"的社会责任，遵循"明德、博学、尚行"的校训精神，配合澳门特区政府"1+4"产业多元发展策略及深合区发展政策。

学院官方介绍中明确列出以下学位体系：

| 层级 | 官方公开课程体系 |
|---|---|
| 本科 | 社会工作学士（BSW） |
| 硕士 | 社会工作硕士（MSW）、应用心理学硕士（MAP）、智慧养老与健康管理硕士（规划中） |
| 博士 | 应用心理学博士（DAP） |
| 未来规划 | 进一步拓展健康领域相关课程 |

### 1.2 学院面向学生的核心卖点

从学生利益角度，FH 的公开资料中可提炼出以下要点：

1. **全球首家大健康学院**：学院定位独特，专注于大健康产业链的人才培养。
2. **本硕博贯通路径清晰**：本科可衔接 MSW、MAP、DAP 或其他相关研究生学习。
3. **两大核心领域**：社会工作（创新社会工作系）与应用心理学（心理分析研究院）并重发展。
4. **国际化师资**：教师团队具备海外学位和前沿研究背景，部分教师来自葡萄牙、美国、英国、澳洲等地。
5. **实践导向**：强调理论研究与实际应用相结合，提供实习、督导、国际交流等机会。
6. **科研平台完善**：设有心理分析研究院、多个研究中心和实验室，支持学生参与科研项目。
7. **区域需求对接**：课程设置紧贴澳门、内地及周边区域在大健康领域的重大需求。

### 2.2 对学生利益影响最大的规则

1. **入学年份规则优先**：课程结构、毕业要求和学分要求应以学生本人入学年度的课程计划/学生手册为准。
2. **实习是社工专业的重要组成部分**：BSW 和 MSW 学生需完成规定时数的实习，并参加实习督导。
3. **论文/毕业项目是硕士/博士毕业核心**：需提前规划论文选题、导师匹配和写作进度。
4. **老年学与社会服务管理是 MSW 的特色方向**：学生应在入学后尽快了解并选择适合自己的专业方向。
5. **应用心理学强调中国文化建设与心理分析视角**：MAP 和 DAP 学生需具备中国文化背景知识，并对心理分析、积极心理学等有基本了解。
6. **选修课可能因人数不足不开课**：需准备备选课程。

## 3. 入学与招生：学生应关心什么

### 3.1 本科入学：常见身份类型

澳门城市大学招生通常按身份类型分流，包括但不限于：

- 澳门本地生；
- 中国内地学生；
- 香港、台湾地区学生；

# 第一部分：本科生关切

## 5. 本科课程详解：社会工作学士（BSW）

### 5.1 课程总览

澳门城市大学大健康学院本科课程以"社会工作学士（BSW）"为主轴，旨在培养拥有高专业适应能力的本地化社会工作人才。课程强调科研基础、理论教学与实践手法相结合，推动社会工作往学术化、专业化及国际化发展。

**课程成立时间**：2016年5月17日（注：大健康学院成立于2024年，但社会工作系历史更久）

### 5.2 课程特色

| 特色 | 说明 |
|---|---|
| 重视多元教学方法 | 结合理论、实务、实习、国际交流等多种形式 |
| 全人发展 | 针对学生在社会工作专业上的全人发展 |
| 国际学术交流 | 推动本地及海外实习课程，促进国际学术交流活动 |
| 研究为基础 | 以研究为基础，结合理论、实务经验及社会未来发展来建构课程设计与教学 |
| 前瞻性培育 | 建构国际学术交流平台，培育学生具前瞻性、创意性的社会服务概念 |

### 5.3 学生核心能力

| 核心能力 | 说明 |
|---|---|
| 国际专业水平 | 培养具国际专业水平의社工人才，以应对澳门的社会需要 |
| 知识应用与学术研究 | 强调知识之实际应用，同时着重发展学生的学术研究能力 |
| 多领域专业知识与技能 | 掌握各个社会工作领域的专业知识及技能，包括：老人社会工作、医务社会工作、家庭社会工作、复康社会工作、多元文化社会工作、社会服务管理 |
| 多角度视野 | 能以多角度和广阔视野认识社会工作，并对社会不同情况进行理性和专业分析 |

### 5.4 师资特色

- 师资主要由本地及国内外知名大学的教授组成；
- 各教学人员均具备社会工作及其相关领域丰富的前线实务经验、社会服务管理经验及相当科研学术能力；
- 学院会邀请外地专家开办讲座、研讨会和工作坊，丰富学生的学习历程；
- 重视学生人格和社会工作专业伦理的培育，以关注弱势群体、倡导社会公义及宣扬社工价值为目标。

### 5.5 毕业出路

#### 就业方向

| 领域 | 具体岗位/机构 |
|---|---|
| 公共行政 | 公共行政服务单位 |
| 社会福利 | 各公私立社福机构、医疗单位、企业组织之社工人员 |
| 人力资源 | 各工商机构之人力资源管理人才 |
| 非营利组织 | 各非营利组织之管理人员 |
| 其他相关 | 公共关系、研究分析等性质工作 |

#### 进修方向

- 继续进修深造与社会工作相关之硕士及博士课程。

## 7. 本科转专业

### 7.1 转专业基本规则

根据澳门城市大学通用规则，本科学生在第一学年结束后可申请转专业，但需满足以下条件：

- 已修完第一学年全部必修课程；
- 平均成绩达到转入专业要求（通常以 GPA 为衡量标准）；
- 转入专业有剩余名额；
- 通过转入专业的面试/考核（如适用）。

### 7.2 大健康学院内转专业

目前大健康学院本科仅有社会工作学士一个专业，暂不涉及院内转专业问题。但如果未来开设新专业，学生可申请转入。

### 7.3 学生常见问题（转专业）

| 问题 | 回答要点 |
|---|---|
| 我现在读的是其他学院的专业，可以转到大健康学院吗？ | 可以申请，但需满足转入条件，且名额有限；建议提前与大健康学院课程主任沟通。 |
| 转专业会影响毕业时间吗？ | 可能会；如果转入专业的课程差异较大，可能需要额外修读课程，导致延期毕业。 |
| 转专业申请什么时候提交？ | 通常在每学年结束前；具体时间以教务处通知为准。 |

## 8. 本科荣誉班与科研机会

### 8.1 荣誉班/精英班概述

大健康学院目前尚未设立独立的"荣誉班"或"精英班"制度。但学院鼓励学生参与科研活动，并提供相关支持。

### 8.2 本科生科研机会

大健康学院鼓励本科生参与科研活动，主要途径包括：

## 9. 硕士课程详解：社会工作硕士（MSW）

### 9.1 课程总览

澳门城市大学大健康学院 MSW 课程是**澳门首个以"老年学"及"社会服务管理"为特色的专业学位课程**，致力于培养具国际专业能力的社工高端人才。

**课程特色**：

- 澳门首创：为澳门首个以"老年学"及"社会服务管理"专业为基础的专业学位课程；
- 国际交流平台：建构国际学术交流平台，培育学生具前瞻性、创意性的社会服务概念；
- 区域需求导向：针对本澳、内地及周边区域在该服务领域中的重大需要；
- 专业督导培训：为未来在发展中所需的专业督导人员提供首个培训平台及社工专业上的培训阶梯。

### 9.2 核心竞争力

- 培养具国际专业能力的社工人才，以应对澳门的社会需要；
- 强调学生对所教授知识之实际应用外，也着重发展学生的学术研究能力；
- 让学生深入了解"老年学"及"社会服务管理"两个专业领域的知识及技能；
- 能以多角度和广阔视野认识社会工作，并对社会不同情况进行理性和专业分析。

### 9.3 师资特色

- 课程强调专业本地化发展和国际化交流合作，结合国内外最新资讯与本地资源；
- 师资团队主要由本地及国内外知名大学的教授组成；
- 各教学人员均具备社会工作及其相关各个领域中丰富的前线实务经验、社会服务管理经验及相当的科研学术能力；

## 10. 硕士课程详解：应用心理学硕士（MAP）

### 10.1 课程总览

澳门城市大学大健康学院 MAP 课程的基本框架是**结合中国文化背景与心理学理论**，注重心理分析和积极心理学视角下的健康促进、心理测量和临床心理学应用等。

**课程特色**：

- 结合中国文化背景与心理学理论；
- 注重心理分析（分析心理学/荣格心理学方向）；
- 强调积极心理学视角下的健康促进；
- 涵盖心理测量与临床心理学应用；
- 科研方法 + 实践技能双线培养；
- 强化心理学在实际应用中的转化。

### 10.2 培养目标

毕业生能够成为应用心理学专业人才，以适应社会、文化、教育、医疗、健康、国防、体育等特定职业领域的心理学专业人才需要，并为申请国内外心理学博士以及心理咨询等应用领域的专业资格认证打下基础。

### 10.3 就业/发展方向

- 社会、文化、教育、医疗、健康、国防、体育等领域的心理学专业工作；
- 继续深造：申请国内外心理学博士；
- 专业认证：心理咨询等应用领域的专业资格认证。

| 开题报告 | 通常在第三学期 | 需提交开题报告，并通过开题答辩 |

1. **提前联系导师**：不要等到开题才联系；第一学期就可以开始了解教师研究方向；
2. **选题要具体**：避免选题过大；确保能在规定时间内完成；
3. **定期与导师沟通**：至少每月一次；及时汇报进展和困难；
4. **查重和格式要提前检查**：避免最后时刻出问题；
5. **保留所有表格副本**：答辩后可能需要归档。
